# DOBY!
이 프로젝트는 하나의 플랫폼에서 개발자 커뮤니티와 프로젝트 참여 인원을 모집하는 두 가지 서비스를 모두 이용할 수 있도록 개발된 서비스입니다.

### 📄프로젝트 소개
개발에 관한 다양한 정보를 얻을 수 있고(커뮤니티) 소소한 일상 이야기를 나누며(스몰톡) 사이드 프로젝트 인원을 빠르고 쉽게 구할 수 있는(모임) 서비스.
